The case for conscious consumption

If we want to make the media better, we have to start consuming better media. Information is inherently social, so the consequences of our information habits go way beyond affecting the individual. There are social consequences.

Overview + Description

  • Measure user behavior + habits
  • Analyze trends from this data
  • Prescribe options for "slimming down."

What we want to build

A system that:

  • Analyzes content
  • Score reading habits
  • Presents analysis (data + viz)
  • Prescribes recommendations for future reading habits.

System specs

  • Multiplatform (mobile + tablet)
  • Integration with reading list apps (Pocket + Kippt + Readability, etc.)

Information pyramid

We've brainstormed some preliminary attributes our system will use to determine the healthiness of a user's information consumption:

  • Length of content
  • Type of article (oped / analysis / news brief / cat video?)
  • Source (NYT vs. Buzzfeed)
  • Reading level
  • Topic / category
  • Links (?)

UI + UX considerations

  • Graphs on graphs on graphs. Giving users a visual presentation of their consumption habits is a must.
  • We want to make this fun. We'd like to incorporate some gaming principles:
  • Reading level: You're currently reading at the leve of a second grader. Step it up!
  • Badges for topical expertise: political junkie, tech star, animal lover
  • Reading recommendations + new sources to consider.

Some questions

  • Tracking versus check-ins?
  • How do we make this experience consistent across platforms + not fragmented?
